# Battery Pass example

## Data models used

OWL-version of the [Battery Pass 1.2.0](https://github.com/batterypass/BatteryPassDataModel/tree/main) extracted using [General-Purpose Tool to Convert Asset Administration Shell Templates to RDFS/OWL](https://www.sciencedirect.com/science/article/pii/S1877050926004515) developed at TNO.
- [CarbonFootprintForBatteries.ttl](OWL%20Battery%20Pass%201.2.0/io.BatteryPass.CarbonFootprintForBatteries/CarbonFootprintForBatteries.ttl)
- [Circularity.ttl](OWL%20Battery%20Pass%201.2.0/io.BatteryPass.Circularity/1.2.0/Circularity.ttl)
- [GeneralProductInformation.ttl](OWL%20Battery%20Pass%201.2.0/io.BatteryPass.GeneralProductInformation/GeneralProductInformation.ttl)
- [Labeling.ttl](OWL%20Battery%20Pass%201.2.0/io.BatteryPass.Labels/1.2.0/Labeling.ttl)
- [MaterialComposition.ttl](OWL%20Battery%20Pass%201.2.0/io.BatteryPass.MaterialComposition/1.2.0/MaterialComposition.ttl)
- [PerformanceAndDurability.ttl](OWL%20Battery%20Pass%201.2.0/io.BatteryPass.Performance/1.2.1/PerformanceAndDurability.ttl)
- [SupplyChainDueDiligence.ttl](OWL%20Battery%20Pass%201.2.0/io.BatteryPass.SupplyChainDueDiligence/1.2.0/SupplyChainDueDiligence.ttl)

## Battery Passport examples

For versioning of the Battery Passport instances, we use Semantic Treehouse.

### Static Battery Passport

A DIN-compliant example of the static Battery Passport DPP is available as [version "DIN-compliant-static"](https://dpp.vocabulary-hub.eu/message-model/MessageModelSpec_27d1e4d9-a256-4f9d-9060-b698d3efe3be)

### Dynamic Battery Passport

An example snapshot-based dynamic Battery Passport DPP is available as [version "snapshot-example-dynamic"](https://dpp.vocabulary-hub.eu/message-model/MessageModelSpec_27d1e4d9-a256-4f9d-9060-b698d3efe3be)
